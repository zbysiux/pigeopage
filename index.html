1. Alokacja Pamięci i Sprzęt VGA

Jądro systemu operacyjnego działa w 32-bitowym trybie chronionym (Protected Mode) architektury x86. Poniższa tabela przedstawia mapowanie pamięci zaimplementowane w kodzie źródłowym:

Zasób / Bufor Adres Pamięci (RAM) Typ danych / Rozmiar Opis
VGA_BUFFER 0xB8000 uint16_t* (80x25) Pamięć ekranu tekstowego trybu kolorowego VGA.
notepad_buffer Statyczny RAM jądra char[2048] Trwały bufor przechowujący zawartość aplikacji Notatnika.

2. Rejestry i Mapowanie Portów I/O (Asm Volatile)

Komunikacja ze sprzętem odbywa się za pomocą niskopoziomowych instrukcji procesora inb oraz outb:

  • 0x60 — Data Port: Odczyt skan kodu wciśniętego klawisza (Klawiatura).
  • 0x64 — Status Register: Sprawdzanie bitu wejściowego bufora klawiatury (Polling). Wysłanie komendy 0xFE wymusza Real Hardware Reset (Reboot).
  • 0x70 — RTC Address Port: Wybór rejestru zegara czasu rzeczywistego (np. sekundy, minuty, godziny).
  • 0x71 — RTC Data Port: Odczyt/Zapis danych CMOS w formacie BCD.

3. Referencja API Jądra (Kernel Functions)

Zarządzanie Ekranem i Tekstem

  • void clear_screen(uint8_t color) — Czyszczenie bufora VGA określonym atrybute koloru, reset kursora do (0,0).
  • void put_char_at(char c, uint8_t color, int x, int y) — Bezpośredni zapis znaku i koloru pod konkretny indeks pamięci ekranu.
  • void print_char(char c, uint8_t color) — Strumieniowe wypisanie znaku z obsługą znaków specjalnych (\n, \b).
  • void print_int(int num, uint8_t color) — Funkcja jądra do konwersji i wyświetlania liczb całkowitych bez użycia biblioteki standardowej C.

Sterowniki i Obsługa Stringów

  • char get_key(void) — Blokująca funkcja odpytująca kontroler PS/2 w pętli (Polling) i mapująca tablicę scancodów na ASCII.
  • void show_polish_time(void) — Odczyt z układu RTC, konwersja BCD do BIN oraz formatowanie polskiej strefy czasowej.

4. Słownik Interpretera Komend (CLI Registry)

Główna pętla kernel_main parsuje komendy wprowadzane do bufora tekstowego (maks. 80 znaków). Rejestr obsługuje:

Komenda Typ Działanie systemowe
help System Wyświetla kompletną listę dostępnych procedur.
notepad Aplikacja Przełącza tryb graficzny VGA na niebieskie tło (0x1F) i uruchamia edytor.
chromium Aplikacja Tekstowy emulator sieci, parsuje statyczne mock-up adresy URL.
reboot Hardware Wysyła sygnał resetu do kontrolera klawiatury w celu restartu PC.
exit / shutdown Hardware Wyłącza przerwania i zatrzymuje procesor za pomocą instrukcji cli; hlt.
